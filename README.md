# Utopia Room

In-browser React playground with a Monaco editor and a live preview. It compiles JSX in the browser, renders in an isolated iframe, and includes basic formatting plus a few starter templates.

## What it is for

- Rapidly sketch and test React UI ideas
- Share or iterate on small React snippets without a build step

## How to run

```bash
npm install
npm run dev
```

Optional:

```bash
npm run build
npm run preview
```
